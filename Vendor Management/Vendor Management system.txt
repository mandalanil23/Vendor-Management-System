SECTION 4 — GOOGLE SHEETS INTEGRATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Google Apps Script (google-script.gs) requirements:
  • doGet(e): Return all data as JSON based on ?sheet= param
    - ?sheet=vendors  → Sheet "Vendors" data
    - ?sheet=invoices → Sheet "Invoices" data
    - ?sheet=payments → Sheet "Payments" data
  • doPost(e): Accept JSON body for Create, Update, Delete ops
    - action: "create" | "update" | "delete"
    - sheet: target sheet name
    - data: row object
    - rowId: for update/delete operations
  • CORS headers on all responses
  • Error handling with proper HTTP status codes

Google Sheets structure (3 sheets):
  Sheet 1 "Vendors": VendorID, Name, Category, GSTNo, 
    ContactPerson, Phone, Email, City, State, Status, 
    JoinDate, BankAccount, IFSC, PaymentTerms
  
  Sheet 2 "Invoices": InvoiceNo, VendorID, VendorName, 
    InvoiceDate, DueDate, Items, Amount, GSTAmount, 
    TotalAmount, Status, PaidDate, PaymentRef
  
  Sheet 3 "Payments": PaymentID, VendorID, InvoiceRef, 
    Amount, PaymentMode, BankRef, PaymentDate, 
    ProcessedBy, Remarks

In index.html:
  • CONFIG object at top of <script>: 
    const CONFIG = { SHEET_URL: "PASTE_YOUR_DEPLOYED_URL_HERE" }
  • All data operations go through fetchData(sheet) and postData(action, sheet, data)
  • Fallback to dummy-data.json if SHEET_URL is empty
  • Loading spinner during API calls
  • Error toast if API fails

After all 3 files, include a detailed setup guide covering:

PART A — Local Setup
  1. Save index.html + dummy-data.json in same folder
  2. Open index.html in Chrome (works offline with dummy data)
  3. Verify all 14 modules load correctly

PART B — Google Sheets Setup
  1. Create a new Google Spreadsheet
  2. Create 3 sheets: Vendors, Invoices, Payments
  3. Add the header rows as specified
  4. Go to Extensions → Apps Script
  5. Delete default code, paste google-script.gs content
  6. Save the project with name "Jainco VMS Backend"
  7. Click Deploy → New Deployment
  8. Type: Web App
  9. Execute as: Me
  10. Who has access: Anyone
  11. Click Deploy → Copy the Web App URL
  12. Open index.html → find CONFIG object
  13. Replace "PASTE_YOUR_DEPLOYED_URL_HERE" with copied URL
  14. Save file — now live with real Google Sheets data

PART C — Online Deploy (Free Hosting)
  Option 1 — GitHub Pages:
    1. Create GitHub account → New repository "jainco-vms"
    2. Upload index.html + dummy-data.json
    3. Go to Settings → Pages → Source: main branch
    4. Live at: https://yourusername.github.io/jainco-vms

  Option 2 — Netlify (drag and drop):
    1. Go to netlify.com → Log in
    2. Drag your project folder to Netlify dashboard
    3. Get instant live URL (e.g. jainco-vms.netlify.app)
    4. Optional: Add custom domain in site settings
